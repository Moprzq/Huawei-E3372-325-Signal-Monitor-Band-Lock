What this extension does

It reads Huawei HiLink API from your modem, stores long-term history in IndexedDB, and can change LTE band masks.

Serving cell — current LTE cell: Band / PCI / eNodeB / EARFCN.
Current bands — bands currently allowed by modem configuration.
Startup band — configuration captured when the extension first saw the modem.
Default band — built-in factory/default mask from net-mode-backup.xml: a0080800c5.
Internet drops — moments when internet check failed while the extension was collecting data.

Safety notes

Changing bands can temporarily disconnect the modem. If the modem disappears, unplug/replug it and use Restore default band.

For B3+B7 the mask is 44. For B7 only the mask is 40.
